# Celebrity Rumour Wire

A free-hostable celebrity gossip/news commentary site built with React + Vite.

## Run locally

```bash
npm install
npm start
```

## Build for Cloudflare Pages

```bash
npm run build
```

Cloudflare settings:

- Framework preset: Vite
- Build command: npm run build
- Build output directory: dist

## Important legal/editorial notes

Do not copy full articles or private content. Link to sources, use short summaries, add original commentary, and label rumours clearly.

## Monetisation

Start with original articles and commentary. Add Google AdSense only after the site has enough high-quality original content and privacy/terms pages.
