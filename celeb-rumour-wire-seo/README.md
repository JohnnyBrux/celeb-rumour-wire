# Celeb Rumour Wire SEO Version

This version adds:

- SEO title and meta description
- Open Graph and Twitter tags
- canonical URL
- robots.txt
- sitemap.xml
- site.webmanifest
- SEO landing pages:
  - /latest-celebrity-rumours
  - /celebrity-breakups
  - /reality-tv-drama
  - /taylor-swift-rumours
  - /kardashian-news
- Story detail URLs under /story/story-title
- NewsArticle JSON-LD schema on story pages
- Improved privacy and terms pages

## Cloudflare Pages settings

Framework preset: Vite
Build command: npm run build
Build output directory: dist

## After uploading to GitHub

Cloudflare should redeploy automatically. Then submit this sitemap in Google Search Console:

https://celeb-rumour-wire.pages.dev/sitemap.xml

## Important SEO note

The site will rank better when you add original commentary and unique pages, not just copied article snippets. Keep source links and avoid presenting unconfirmed gossip as fact.
